# Mem0 Shell Aliases

**Last Updated:** 2026-02-19

## Available Aliases

| Alias | Usage | Description |
|-------|-------|-------------|
| `m0` | `m0 'your query'` | Query mem0 memories |
| `m0oliver` | `m0oliver` | Query Oliver-related memories |
| `m0pending` | `m0pending` | Show pending tasks/memories |
| `m0prep` | `m0prep 'name'` | Prep memories for a named context |

These aliases are loaded from the intel-system shell configuration. To reload:

```bash

# or
source ~/.bash_profile
```
